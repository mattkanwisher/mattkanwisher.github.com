Driftwood.js a simple javascript logging framework

Jul 22nd, 2012

We were joking around the office the other day and were wondering why there weren’t any simple javascript logging libraries. The ones out there tried to completely copy log4j with abstract classes. We just wanted something that could have configurable log levels. Secondarily we needed to be able to post errors to the server.

Here are a few examples from Driftwood, all you need to do is set your environment and then start logging!

If you want exceptions from users to be posted to your server just simply call the exception method. We automatically catch any window.onerrors and send them.

So we cleaned up the internal code and added a nice suite of jasmine unit test. Now you can fork it on gitubhub Driftwood.js.
